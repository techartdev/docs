There are different ways to install SCP2P depending on whether you want the graphical Desktop App or the command-line core utilities (`scp2p-cli` and `scp2p-relay`).

## Desktop App (Recommended)

For most users, the easiest way to get started is by downloading the pre-built desktop application. 

Go to the [GitHub Releases page](https://github.com/techartdev/scp2p/releases) and download the appropriate installer for your OS:
- **Windows:** `.msi` or `.exe`
- **macOS:** `.dmg` or `.app.tar.gz`
- **Linux:** `.AppImage` or `.deb`

---

## Command Line Utilities (CLI & Relay)

If you are setting up a server, a headless node, or prefer the command line, you can install the `scp2p-cli` and `scp2p-relay` binaries using our 1-line installation scripts. These scripts automatically detect your operating system and CPU architecture, download the latest pre-compiled release, and place the binaries in your system's `PATH`.

### Linux & macOS

Open your terminal and run:

```bash
curl -sSfL https://raw.githubusercontent.com/techartdev/scp2p/main/install.sh | sh
```

- **With `sudo` access:** Installs globally to `/usr/local/bin`.
- **Without `sudo` access:** Installs locally to `~/.local/bin`.

*You can pin a specific version or change the destination directory:*
```bash
curl -sSfL https://raw.githubusercontent.com/techartdev/scp2p/main/install.sh | sh -s -- --version 0.1.0 --dir ~/bin
```

### Windows

Open **PowerShell** or **Windows Terminal** and run:

```powershell
irm https://raw.githubusercontent.com/techartdev/scp2p/main/install.ps1 | iex
```

- Installs to `%LOCALAPPDATA%\scp2p-relay\` (or a similar local app data folder).
- Automatically adds the installation directory to your user `PATH`.

*You can pin a version or change the install directory using environment variables before running the script:*
```powershell
$env:SCP2P_VERSION="0.1.0"
$env:SCP2P_INSTALL_DIR="C:\tools\scp2p"
irm https://raw.githubusercontent.com/techartdev/scp2p/main/install.ps1 | iex
```

## Pre-built Binaries (Manual Download)

If you prefer not to use the installation scripts for the CLI tools, you can download the standalone binaries directly from the [GitHub Releases page](https://github.com/techartdev/scp2p/releases).

Available architectures:
- `linux-x86_64`
- `linux-aarch64` (Raspberry Pi, AWS Graviton)
- `macos-x86_64` (Intel Macs)
- `macos-aarch64` (Apple Silicon M1/M2/M3)
- `windows-x86_64.exe`

---

## Building from Source

If you want to build the tools from source, ensure you have the **Rust toolchain** installed (Rust Edition 2024).

```bash
git clone https://github.com/techartdev/scp2p.git
cd scp2p

# Install the CLI
cargo install --path crates/scp2p-cli

# Install the Relay node
cargo install --path crates/scp2p-relay
```
