While the `scp2p-cli` provides full access to the core protocol, the easiest way for non-developers to use SCP2P is via the desktop application.

The desktop app provides a user-friendly React interface built on top of the SCP2P Rust core using [Tauri](https://tauri.app/).

## Prerequisites

To build and run the desktop app from source, you need:

1. **Rust toolchain** (Edition 2024)
2. **Node.js** (v18+) and `npm`
3. **Tauri OS dependencies**:
   - On Windows: Build Tools for Visual Studio
   - On macOS: Xcode Command Line Tools
   - On Linux: `libwebkit2gtk-4.1-dev`, `build-essential`, `curl`, `wget`, `file`, `libxdo-dev`, `libssl-dev`, `libayatana-appindicator3-dev`, `librsvg2-dev`

*(See the official [Tauri Prerequisites guide](https://tauri.app/v1/guides/getting-started/prerequisites) for detailed instructions for your OS).*

## Running in Development Mode

The desktop app source code is located in the `app/` directory of the workspace.

<Steps>
  <Step title="Navigate to the app directory">
    ```bash
    cd app
    ```
  </Step>
  <Step title="Install Node dependencies">
    Install the required React and UI packages.
    ```bash
    npm install
    ```
  </Step>
  <Step title="Start Tauri dev server">
    This command will start the Vite frontend development server, compile the Rust backend, and open the native window.
    ```bash
    npm run tauri dev
    ```
    *Note: The first time you run this, compiling the Rust backend may take a few minutes.*
  </Step>
</Steps>

## App Features

The current prototype of the desktop app supports the core SCP2P workflows:

- **Identity Management:** View your node ID and manage your cryptographic keys.
- **Publishing:** Drag and drop folders to create new public or private Shares.
- **Subscriptions:** Paste an Invite Link or Share ID to subscribe to a peer's catalog.
- **Local Search:** Instantly search across all your subscribed content.
- **Downloads:** Click to initiate swarmed, verified downloads from available peers.

## Architecture of the App

The desktop app is designed as a thin frontend over the robust core library.

- **Frontend:** React + TypeScript + TailwindCSS. Located in `app/src/`.
- **Backend/Core:** Rust using the `scp2p_core::NodeHandle` API. The Tauri commands are located in `app/src-tauri/src/main.rs`.
- **Persistence:** Uses SQLite for data storage, managed by the Rust core, ensuring consistency whether the user is running the GUI or the CLI.
